## a) Utility Class

**ตัวอย่าง:** `System.Math`

`System.Math` จัดอยู่ในกลุ่ม Utility Class เพราะ:

- **ไม่สามารถสร้าง instance ได้** – คลาสนี้เป็น `static` ทั้งหมด ไม่มี constructor ให้เรียกใช้ `new Math()` ได้
- **เมธอดทั้งหมดเป็น static** – เช่น `Math.Sqrt(16)`, `Math.Pow(2, 3)`, `Math.Abs(-5)` เรียกใช้ผ่านชื่อคลาสโดยตรงโดยไม่ต้องสร้าง object
- **ไม่มี state ภายใน** – คลาสนี้ไม่มี field หรือ property ที่เก็บค่าภายในตัวเอง ทำหน้าที่เป็นเพียง "กล่องเครื่องมือ" ที่รวมฟังก์ชันคณิตศาสตร์ไว้ให้ใช้งาน
- **ทำงานแบบ input/output เท่านั้น** – รับค่า parameter เข้าไป แล้วคืนผลลัพธ์กลับมา โดยไม่มีผลข้างเคียง (side effect) ใด ๆ

## b) Mutable Class

**ตัวอย่าง:** `System.Text.StringBuilder`

`System.Text.StringBuilder` จัดอยู่ในกลุ่ม Mutable Class เพราะ:

- **สามารถแก้ไข state ภายในได้หลังจากสร้าง object แล้ว** – เมื่อสร้าง `StringBuilder` ขึ้นมา สามารถเปลี่ยนแปลงเนื้อหาข้อความภายในได้ตลอดเวลาผ่านเมธอดต่าง ๆ
- **มีเมธอดที่เปลี่ยนแปลงข้อมูลภายใน** – เช่น `Append()` เพิ่มข้อความต่อท้าย, `Insert()` แทรกข้อความ, `Remove()` ลบข้อความบางส่วน, `Replace()` แทนที่ข้อความ ซึ่งทุกเมธอดเหล่านี้แก้ไข object เดิมโดยตรง
- **ไม่สร้าง object ใหม่ทุกครั้งที่แก้ไข** – ต่างจาก `String` ที่ต้องสร้าง object ใหม่ทุกครั้ง `StringBuilder` ทำงานกับ buffer ภายในเดิมเพื่อประสิทธิภาพที่ดีกว่า
- **เหมาะสำหรับการต่อ string จำนวนมาก** – เพราะ mutable จึงไม่ต้องจองหน่วยความจำใหม่ซ้ำ ๆ

## c) Immutable Class

**ตัวอย่าง:** `System.String`

`System.String` จัดอยู่ในกลุ่ม Immutable Class เพราะ:

- **เมื่อสร้าง object แล้ว ค่าภายในจะเปลี่ยนแปลงไม่ได้** – เมื่อกำหนดค่า string เช่น `string s = "Hello";` ข้อความ `"Hello"` จะไม่สามารถถูกแก้ไขได้โดยตรง
- **ทุกการดำเนินการที่ดูเหมือนแก้ไข จะสร้าง object ใหม่** – เช่น `s.ToUpper()` ไม่ได้เปลี่ยน `s` เป็นตัวพิมพ์ใหญ่ แต่จะ return `String` ใหม่ที่เป็นตัวพิมพ์ใหญ่ออกมาแทน ค่าเดิมของ `s` ยังคงเป็น `"Hello"`
- **ข้อดีของ Immutability คือความปลอดภัย** – เนื่องจากค่าไม่เปลี่ยนแปลง จึงสามารถแชร์ `String` ข้ามหลาย thread ได้อย่างปลอดภัยโดยไม่ต้องกังวลเรื่อง race condition
- **String interning** – .NET ใช้กลไก string interning ที่เก็บ string ที่มีค่าเหมือนกันไว้ที่เดียวในหน่วยความจำ ซึ่งทำได้เพราะ string เป็น immutable จึงรับประกันได้ว่าค่าจะไม่ถูกเปลี่ยนโดยใครทั้งสิ้น
